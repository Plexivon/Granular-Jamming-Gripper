# Granular-Jamming-Gripper
In a nutshell, This project explores the use of a soft, flexible gripper instead of a traditional robotic claw. It molds around objects when soft, then a vacuum makes it stiff so it can securely hold and move them.

This project is a robotic manipulator based on reversible phase transition. Functionally, it serves as an alternative to a traditional robotic claw, acting as the end effector of a robotic arm used to handle a variety of payloads. Rather than gripping or squeezing objects directly, the manipulator utilizes a granular jamming mechanism. When at atmospheric pressure, (The default state), the material inside the manipulator remains flexible and easily deform-able, allowing it to conform to the shape of an object. When a vacuum is applied to the enclosing membrane, inter-granular friction increases, causing the material to transition into a rigid state. This process; referred to as reversible phase transition, enables the manipulator to securely hold its shape and effectively grasp and manipulate the object.
